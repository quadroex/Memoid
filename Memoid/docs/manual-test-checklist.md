# Memoid: checklist ручного тестування

## A. Запуск

- Відкрити `https://localhost:7113`.
- Переконатися, що темна головна сторінка завантажилась.
- Відкрити `https://localhost:7113/openapi/v1.json`.
- Переконатися, що OpenAPI JSON доступний.

## B. Категорії

- Створити категорію з коректною назвою та описом.
- Переконатися, що категорія з'явилась у списку.
- Переконатися, що категорія доступна у формі створення шаблону.
- Відредагувати назву категорії.
- Відредагувати опис категорії.
- Змінити активний статус категорії.
- Видалити або деактивувати категорію.
- Перевірити, що список категорій оновився без перезавантаження сторінки.

## C. Шаблони

- Додати шаблон із коректним JPG, PNG або WEBP зображенням.
- Вказати назву шаблону.
- Обрати активну категорію.
- Переконатися, що шаблон з'явився у списку.
- Переконатися, що зображення шаблону відображається.
- Натиснути `Використати в редакторі`.
- Переконатися, що шаблон завантажився у canvas.

## D. Редактор

- Завантажити власне зображення.
- Вибрати існуючий шаблон.
- Ввести верхній текст.
- Ввести нижній текст.
- Змінити шрифт.
- Змінити розмір тексту.
- Змінити колір тексту.
- Змінити позицію тексту.
- Застосувати ефект `Чорно-білий`.
- Застосувати ефект `Контраст`.
- Застосувати ефект `Червоний фільтр`.
- Застосувати ефект `Легке розмиття`.
- Застосувати ефект `Шакалізація`.
- Перемкнути режим кадрування `Оригінальні пропорції`.
- Перемкнути режим кадрування `Вписати повністю`.
- Перемкнути режим кадрування `Заповнити полотно`.
- Перемкнути режим кадрування `Квадратний мем`.
- Завантажити готовий мем.
- Зберегти мем у галерею.

## E. Галерея

- Переконатися, що збережений мем з'явився у галереї.
- Відкрити preview мема.
- Закрити preview кнопкою.
- Закрити preview клавішею Escape.
- Завантажити мем із картки.
- Завантажити мем із preview.
- Додати мем в улюблені.
- Прибрати мем з улюблених.
- Увімкнути фільтр `Лише улюблені`.
- Вимкнути фільтр `Лише улюблені`.
- Видалити тестовий мем із галереї.
- Переконатися, що галерея оновилась без повного перезавантаження сторінки.

## F. Негативні тести

- Спробувати створити категорію з порожньою назвою.
- Спробувати створити категорію з дубльованою назвою.
- Спробувати додати шаблон без назви.
- Спробувати додати шаблон без категорії.
- Спробувати додати шаблон без файлу.
- Спробувати додати файл шаблону з непідтримуваним типом.
- Спробувати завантажити зображення більше 5 МБ.
- Спробувати зберегти мем без вибраного шаблону або власного зображення.
- Спробувати завантажити готовий мем без зображення у редакторі.
- Спробувати відкрити зображення, файл якого фізично відсутній, і перевірити placeholder.
